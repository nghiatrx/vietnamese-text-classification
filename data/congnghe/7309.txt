Windows XP vẫn là hệ điều hành phổ biến thứ ba trên thế giới, macOS không có chân trong tốp 3 Rất nhiều người dùng máy tính trên toàn thế giới vẫn gắn bó với Windows XP bất chấp việc hệ điều hành này đã quá cũ và không còn được Microsoft hỗ trợ.Tin tặc vừa lợi dụng một lỗ hổng lớn trong Windows XP để tung ra một cuộc tấn công trên mạng đang gây hỗn loạn trên toàn cầu. Tuy nhiên, hệ điều hành này vẫn đang chạy trên rất nhiều máy tính trên toàn thế giới dù Microsoft đã ngừng hỗ trợ bảo mật cho nó vào ngày 8/4/2014. Theo thống kê từ Net Applications, Windows XP hiện vẫn đang là hệ điều hành phổ biến thứ ba trên thế giới, chiếm tới 7,04% thị phần. Điều đó có nghĩa là một hệ điều hành lạc hậu, không còn được hỗ trợ đang phổ biến hơn mọi phiên bản Windows 8, macOS và Linux. Xếp phía trên Windows XP là Windows 7 và Windows 10 với thị phần 48,5% và 26,28% tương ứng. Dưới đây là tốp 5 hệ điều hành phổ biến nhất toàn cầu theo Net Applications: 1. Windows 7: 48,5% 2. Windows 10: 26,28% 3. Windows XP: 7,04% 4. Windows 8.1: 6,96% 5. macOS X 10.12: 3,21% Vậy có bao nhiêu máy tính trên thế giới vẫn đang chạy Windows XP? Theo hãng phân tích Gartner, tính tới năm 2014 có khoảng 2 tỷ máy tính đang được sử dụng trên toàn cầu. Gartner vẫn chưa đưa ra thống kê cho năm 2017 nhưng nếu tính theo con số 2 tỷ thì ít nhất khoảng 140 triệu máy tính vẫn đang chạy Windows XP. Europol, cảnh sát châu Âu, cảnh báo ràng cuộc tấn công an ninh mạng "WannaCry:" sẽ tiếp tục hoành hành trong tuần này khi người dùng trở lại công sở và mở máy tính cá nhân của họ. WannaCry là một mã độc tống tiền, nó mã hóa những dữ liệu quan trọng trên máy tính của người dùng và đòi tiền chuộc. Theo Europol, WannaCry hiện có ít nhất 200.000 nạn nhân trên 150 quốc gia và gây hỗn loạn cho nhiều tổ chức trên toàn cầu. Mặc dù không còn hỗ trợ Windows XP nữa nhưng cuối tuần trước Microsoft vẫn buộc phải tung ra một bản vá lỗi khẩn cấp cho Windows XP, Windows 8 và Windows Server 2003.
