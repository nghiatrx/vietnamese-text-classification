Windows 10 được công ty bảo mật công nhận an toàn hơn Windows 7 Kể từ khi Windows 10 ra mắt vào năm 2015, Microsoft đã luôn quảng cáo rằng đây là bản Windows tốt nhất về mặt bảo mật. Tuy nhiên, cho tới nay điều này mới được một hãng nghiên cứu độc lập công nhận.Theo báo cáo mới được công bố của Webroot về malware trong năm 2017, Windows 10 đang thể hiện đúng như những gì Microsoft quảng cáo. Báo cáo này cho thấy một sự khác biệt đáng kể giữa malware trên Windows 7 và Windows 10. "Trong năm 2017 có ít malware hơn so với năm 2016, tuy nhiên các con số sẽ đáng chú ý hơn khi xem xét dựa trên các hệ điều hành. Chỉ 15% số files được xác định là malware trong năm 2017 xuất hiện trên Windows 10, trong khi đó 63% xuất hiện trên Windows 7, một hệ điều hành phổ biến khác được dùng trong môi trường doanh nghiệp. Trên Windows 10, chúng tôi thấy trung bình có 0,04 file malware trên một thiết bị, bằng một nửa so với con số 0,08 trên Windows 7", Bên cạnh đó, hãng này cũng chỉ ra rằng với người tiêu dùng thông thường, Windows 10 an toàn hơn hai lần so với Windows 7. "Số malware trên một thiết bị Windows 10 sử dụng trong gia đình chỉ là 0,07 so với 0,16 trên Windows 7 và 0,17 trên Windows XP. Giống như các thiết bị dành cho doanh nghiệp, Windows 10 cũng an toàn hơn hai lần so với Windows 7 dành cho các thiết bị trong gia đình. Số lượng malware trên một thiết bị vẫn không đổi trong suốt năm 2017, trung bình 0,55 file trên một thiết bị", Ngoài ra, báo cáo của Webroot cũng tập trung vào cuộc tấn công WannaCry làm tê liệt gần 200.000 máy tính ở hơn 100 quốc gia trên toàn cầu trong năm vừa qua. Mặc dù WannaCry nhắm mục tiêu vào máy tính Windows nhưng tin vui là nó không có bất kỳ ảnh hưởng nào trên Windows 10. Hầu hết tất cả các thiết bị trở thành nạn nhân của WannaCry đều chạy Windows 7 và chỉ riêng cuộc tấn công đó đã gây thiệt hại 4 tỷ USD cho các doanh nghiệp. Nhìn chung, báo cáo của Webroot là một tin vui với Microsoft và khuyến khích mọi người chuyển sang sử dụng Windows 10. Với người dùng đang chạy Windows 7, hãy đảm bảo rằng hãy nâng cấp sớm lên Windows 10 trước khi hệ điều hành này bị kết thúc hỗ trợ vào tháng 1/2020.
