GitHub phát hành nền tảng Marketplace cho phép các nhà phát triển tìm những công cụ mình cần GitHub vừa tung ra dịch vụ Marketplace giúp các nhà phát triển dễ dàng tìm công cụ phù hợp để cải tiến và tinh chỉnh luồng công việc.Theo thông tin đăng tải trên blog công ty, GitHub đã trình làng nền tảng Marketplace cho phép thành viên dễ dàng khám phá và mua ứng dụng cần thiết phục vụ các dự án phát triển phần mềm, từ tích hợp liên tục (ontinuous integration), quản lý dự án cho tới việc xem xét, đánh giá code. GitHub vừa trình làng nền tảng mới mang tên Marketplace Nền tảng mới tạo điều kiện cho nhà phát triển nhanh chóng tiếp cận với những công cụ mới mà không cần tạo nhiều tài khoản hoặc thiết lập các phương thức thanh toán riêng. Marketplace được chia thành 5 mục khác nhau, gồm Code quality, Code review, Continuous integration, Monitoring và Project management. Ngoài ra, nền tảng còn hỗ trợ các ứng dụng phổ biến như Travis CI, Sentry và Waffle. GitHub hứa hẹn sẽ cung cấp thêm nhiều ứng dụng trong thời gian sớm nhất. Mục đích xa hơn là cho phép các nhà phát triển có cơ hội tải ứng dụng riêng nếu đáp ứng được các tiêu chuẩn cộng đồng. Công ty còn đặt lại phần Integrations thành GitHub Apps, trở thành nơi cho phép nhà phát triển kiểm soát nhiều hơn khả năng cấp quyền và truy cập vào kho lưu trữ. Cuối cùng, GitHub đã quyết định cung cấp hàm API của ngôn ngữ truy vấn GraphQL cho tất cả mọi người. Nhờ đó bạn có thể tạo ra các công cụ riêng với khả năng truy cập lượng dữ liệu lớn. Đây là tin vui cho giới lập trình, hãy thử Marketplace theo đường dẫn
