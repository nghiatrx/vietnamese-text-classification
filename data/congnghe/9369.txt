Microsoft vượt Facebook trở thành doanh nghiệp đóng góp nhiều mã nguồn mở nhất thế giới Trong suốt những năm 90 và 2000, Microsoft đã cạnh tranh mạnh mẽ với khái niệm mã nguồn mở - những phần mềm miễn phí được phát triển bởi các nhóm lập trình viên tình nguyện ở khắp nơi trên thế giới.Tuy nhiên, kể từ khi Satya Nadella nhận chức CEO vào năm 2014, quan điểm của Microsoft đã thay đổi. Microsoft tỏ vẻ thân thiện với mã nguồn mở và thậm chí còn hỗ trợ hệ điều hành nguồn mở Linux trên nền tảng điện toán đám mây Microsoft Azure. Thậm chí, theo thống kê của GitHub, Microsoft đã vượt qua Facebook để trở thành doanh nghiệp có số nhân viên đóng góp nhiều code nhất cho các dự án mã nguồn mở. GitHub là trung tâm của thế giới mã nguồn mở và còn được biết tới với cái tên "Facebook của các lập trình viên". Hiện Microsoft đang xếp thứ nhất với 16.419 nhân viên đóng góp code cho các dự án mã nguồn mở. Trong khi đó Facebook đứng thứ hai với 15.682 nhân viên. Đây là cột mốc quan trọng với Microsoft trong cuộc chiến tranh giành khách hàng điện toán đám mây với Amazon. Phần lớn các doanh nghiệp lớn và nhỏ đều sử dụng ít nhất một vài chương trình mã nguồn mở nhằm đáp ứng nhu cầu của thời đại internet. Linux, đối thủ lớn nhất của Microsoft, hiện trở thành hệ điều hành chuẩn mực cho rất nhiều hệ thống máy chủ doanh nghiệp dù chẳng mấy khi thấy nó được cài đặt trên máy tính cá nhân. Vì vậy, quay sang ủng hộ mã nguồn mở trở thành trách nhiệm của Mcirosoft nhằm thuyết phục các CIO của các doanh nghiệp chuyển cơ sở hạ tầng máy chủ của họ lên nền tảng điện toán đám mây Microsoft Azure. Đây chính là lý do tại sao Nadella tuyên bố rằng Microsoft yêu Linux và tại sao Microsoft lại trình làng một số công nghệ chủ chốt dưới dạng mã nguồn mở. Việc Microsoft trở thành doanh nghiệp đóng góp nhiều mã nguồn mở nhất thế giới thể hiện rằng họ đang thực hiện những gì mình đã nói. Dù sao thì đây cũng là một bước tiến lớn của Microsoft dù GitHub chỉ thống kê qua những dự án công khai bởi còn rất nhiều hãng sử dụng GitHub một cách bí mật.
